# CS-FUNDAMENTALS

## Project Title
Calculator & Python Assignments

## Description
This project includes basic Python programs such as a calculator, file handling, CSV handling, exception handling, and more.

## Installation Steps
1. Clone the repository using git clone.
2. Install Python 3.x
3. Run the project files using python filename.py

## How to Run the Program
Open terminal and type:
python CALCULATOR.py
